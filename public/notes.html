Dec 22 2022

Open Payment Host

Open Payment Host is a self-hosted, minimalist, FOSS payments host.
It gives you control over your sales and helps you save money by avoiding commissions to payment hosts.
Disclosure: Open Payment Host is built by me.

 

Cactus Comments

Cactus Comments is a federated comment system for the web, based on the Matrix protocol.
A good choice for privacy respecting websites.

 

Shotcut

Shotcut is a free, open source, cross-platform video editor.
A great choice for those who want a minimalist open-source video editor with hardware acceleration.

Dec 19 2022

PocketBase

PocketBase is a self-hosted, open-source alternative for firebase.
It uses SQLite and built with Go, So its portable and can be easily integrated within applications.

 

BUILD LIST

Build List is a directory of on-demand manufacturers for your creative projects from N-O-D-E.
Each site listed has instant quotes / online ordering so you can upload your project files and immediately see how much it will cost and how long it will take to make.

 

REFACTORING GURU

Refactoring guru website contains programming patterns, Specifically refactoring and design patterns.
The concepts are explaining using graphics aimed at visual learners.
The website is run by an indie and is sustained using certificate courses.
